# __Login, Dashbordand, Registration application__
* A simple login form where users will login or register to navigate dashboard of list of students.<br/>
## __Registration page__
* Lets users input their information if the user has no account.<br/>
## __Login page__
* Users input their account details to login.<br/>
## __Dashboard page__
* Users will be brought to the Dashboard where the Students data can be seen.<br/>
## __To use the application__
<div align="justify">
    You will first be brought to the login page, where you will input your account details such the username and the password, then press the login button to go to the dashboard page. If you don't have account yet, you need to click the link "No account yet? Click here to register", to create your account.

    In the registration page, input your account information, such Name, Username, Email, Password, and the Confirmed password. Then press register to create your account. You can click "Already have an account? Login here" if you already have an account.

    Lasty, if you successfully login, you will be brought to the dashbaord page.
</div>
